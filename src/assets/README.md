# Images

Real project photography has been wired in directly under `public/images/` and
referenced from `src/data/siteData.js` and the page files (Home, About, Project,
Amenities, FloorPlans, Gallery, Contact). The Location page still uses a
placeholder for the map (no map image was provided).

To swap any image, replace the file in `public/images/` (keep the same filename)
or update the path in `src/data/siteData.js` / the relevant page file.

The Project page's walkthrough video is at `public/images/video.mp4`
(compressed from the original for reasonable file size/load time).
